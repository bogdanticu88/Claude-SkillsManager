## Red Flags: What Could Go Wrong

1. **Anthropic/OpenAI launches competing store**
   - Mitigation: Multi-LLM from start, open-source model, community governance
   - Speed: Move fast, ship Phase 1 in 6 weeks, Phase 2 in 8 weeks

2. **Can't recruit skilled creators for bootstrap**
   - Mitigation: Build 5-10 skills yourself, recruit top creators via personal network
   - Budget impact: +$10k and +4 weeks

3. **Manifest schema design flaw discovered mid-Phase 1**
   - Mitigation: Validate schema with 5 test skills in Week 1
   - Fix before Week 2 starts (1-week buffer)

4. **GitHub access tokens get stolen (security incident)**
   - Mitigation: Use GitHub-managed tokens, rotate frequently, monitor for abuse
   - Response: Public transparency, revoke tokens, security audit

5. **Creator backlash on monetization in Phase 2**
   - Mitigation: Transparent communication, opt-in features, community feedback loop
   - Response: Adjust pricing, communicate rationale
