About SkillPM

SkillPM is an open-source package manager for LLM skills. It provides a registry for discovering, installing, and managing skills that extend the capabilities of large language models like Claude, GPT-4, Gemini, and locally-hosted models.

Install the CLI

go install github.com/skillpm/skillpm@latest

Quick Start

# Register an account
skillpm auth register --username myname --email me@example.com

# Search for skills
skillpm search "code review"

# Install a skill
skillpm install code-reviewer

# Run a skill
skillpm run code-reviewer ./myfile.py
